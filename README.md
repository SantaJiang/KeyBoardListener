# 简介
* 这是一款采用Qt5开发的键盘按键检测软件，主要用来检测键盘坏键，也可以用于学习Hook以及QKeyEvent相关知识。
* 软件代码已经尽量简化，下载后编译时Kits采用MinGW 64-bit即可正常编译。
